## 🛠️ Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/AlejoAmaris03/porfolio-alejo-amaris.git
   cd porfolio-alejo-amaris-main

2. **Install dependencies**
   ```bash
   npm install

3. **Run the project**
   ```bash
   ng serve --open

4. The app should be running at: http://localhost:4200
